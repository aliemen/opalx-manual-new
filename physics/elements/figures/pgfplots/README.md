# PGFPlots Sources

These files are copied from the external `bmad-opalx` benchmark workspace and are the local source-of-truth for the bend comparison figures used in the Physics Manual.

Usage:
- HTML uses the rendered PNG assets in the parent `figures/` directory.
- PDF currently uses the vector PDF assets in the parent `figures/` directory to preserve Quarto figure numbering and cross references.
- The `.tex` and `.csv` files here allow the plots to be rebuilt locally or migrated later to raw LaTeX inclusion if the figure-reference workflow is refactored.
